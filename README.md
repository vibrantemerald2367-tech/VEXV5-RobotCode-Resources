# VEX Robotics V5 Programs

This repository contains VEX robotics competition code for both autonomous and driver control periods. These programs were developed for various VEX competition seasons and showcase different robotic functionalities and strategies.

## 🤖 Autonomous Programs (please note that individual number values may vary based on individual motor inaccuracies, use this as reference only)

### Features
- **Pre-programmed routines** for competition matches
- **Sensor integration** (GPS, vision, inertial, distance sensors)
- **Object manipulation** sequences (lifting, intake, scoring)
- **Field navigation** with position tracking

### Key Autonomous Routines
- **Skills Challenge** - Complete 60-second routines
- **Match Auton** - 15-second match opening routines
- **Testing Modes** - Individual component testing

## 🎮 Driver Control

### Features
- **Button mapping** for complex mechanisms
- **Mode toggles** for different match needs 

## 🛠️ Technical Details

### Platform
- **VEX V5 Brain** - Primary controller
- **VEXcode V5** - Development environment
- **Blocks** - Programming type (although also offers Python and C++)

### Sensors (some used, some not)
- Optical sensors for object colour detection
- Inertial sensors for orientation
- Distance sensors for navigation
- GPS for field positioning

### Motor Control
- PID loops for accurate movement
- Slew rate control for smooth acceleration
- Motor groups for easier control
- Custom gear ratio calculations

## 🚀 Getting Started

### Prerequisites
- VEX V5 Robot Brain
- VEXcode V5 software
- Compatible VEX materials (you will have to build your own becuase I am not sharing my designs here)

### Setup
1. Clone this repository
2. Open projects in VEXcode V5
3. Configure robot-specific settings in `robot_config.h`
4. Upload to VEX brain and test

### Configuration
Varies for each individual robot (refer to README file of each competition season where the specifics are listed)
